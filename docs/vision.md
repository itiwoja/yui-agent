# Yui Cloud Agent — ビジョン（MVPスコープ外・ロードマップ用）

> ここに書かれたものは 7/12 提出の MVP には**実装しない**。README/ProtoPedia の展望として語り、ハッカソン後に新しい Intent として立てる。

## 最終形: おしゃべりできるAI秘書アバター

- **壁打ち・アイデア出し**: 対話しながらブレストの相手になる。文脈からユーザーの意図・優先事項を推測する
- **裏で調査を続ける**: 会話に出たトピックをバックグラウンドで調査し、裏どりした情報を次の会話で持ってくる（ローカルYuiChanの北極星④=自律的好奇心探索がクラウドでスケールした形）
- **不在時の検証**: ユーザーが使っていない間も、過去に出た案・アイデアを検証し、結果を溜めておく
- **アバター**: ローカル版YuiChan（デスクバディ）と人格・記憶を共有し、ローカル/クラウドのハイブリッドで動く

## MVP（7/12提出）はその第一歩

「入力されなかったタスクの発見」= 秘書の能力の最小単位。対話から意図を汲む・記憶する・自分から指摘する、というコア動作を1本のE2Eで証明する。

## ローカルYuiChanとの関係

- ローカル: 完全オフライン・プライバシー最優先のデスクバディ（qwen3:8b / Ollama）
- クラウド: Gemini + GCPフルマネージドの最強版。記憶アーキテクチャの知見（fact正規化・想起・自律探索）はローカル版から移植する
